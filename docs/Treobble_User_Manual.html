What is Treobble?

Treobble lets you rapidly scroll, select and dial (message and email too) numbers by swiping across the screen.

     

Downloads

The latest version of Treobble can be downloaded here: (PRC | ZIP). The latest version of this user manual can be viewed online here.

Installing Treobble

HotSync or copy Treobble.PRC to your Treo phone. You can launch Treobble from the Treo Home listing.

Uninstalling Treobble

Delete Treobble from the [Home]->[Menu]->[Delete] window.

Compatibility

Treobble should work as described with the Treo 600, Treo 650, Treo 700p, Treo 680, Treo 755p and the Palm Centro Smartphones.

Treobble has been developed and tested on the Cingular Treo 650 (Unlocked) with the latest updater (Updater: 1.17, Firmware: 01.51, Software: Treo650-1b.17-CNG) available at the Palm website.

Operation

Treobble Main Form
Touchscreen-swipe response Swipe (Up or Down) Entries scroll as appropriate.
Touch row (for popup delay time) and leave Selected entry is dialed.
Keypad response 5-Way Up Entries scroll down.
5-Way Down Entries scroll up.
5-Way Left Move row selector up.
5-Way Right Move row selector down.
5-Way Center Selected entry is dialed.
Side Key Selected entry is dialed.
Return Key Selected entry is dialed.
"0" Reset row selector.
"." Reset row selector.
Backspace Show first page.
Shift-Backspace Show last page.
Space Entries scroll up.
"A" to "Z" or "a" to "z" Scroll to first entry with selected character.

Dialer Form
Keypad response 5-Way Navigates or selects buttons or controls as appropriate.
Volume Keys Moves up or down form controls.
Side Button Dials currently selected number.
Backspace Exits dialer.
Space Exits dialer.
Enter/ Return Key Dials currently selected number.


Treobble Preference Settings
Initial Delay Initial Delay (despite meaning something else) is the sampling rate. Values between 3 and 15 milliseconds work well.
Shift Speed-Threshold Number of pixels from the initial touch location where speed shifts occur while swiping up or down.
Reverse Scroll Reverses the scrolling response.
Popup Delay Number of ticks (~ 1/100 second) to tap-and-hold a particular entry to pop-up the dialer. Values around 20 ticks work well.

Features of Treobble

  • Tiny PRC (< 32 KB)
  • NO hooking to 'hede' notifications.

Limitations of Treobble

  • Treobble currently has a limitation of approximately 3000 addressbook entries.
  • May take several seconds to load addressbook data into memory for large databases (> 1500 entries). This should not be a problem if you have less than 500 entries in your addressbook.
  • It is not possible to do incremental searches in Treobble.

Known Bugs and Issues

  • Some issues with the Centro.
  • [Normal] Issues with less than seven addressbook entries.
